Tabelas com HTML

As tabelas são usadas para exibir dados de forma estruturada. Para inserirmos uma tabela na nossa página usamos a tag <table></table>. Essa tag pode receber o atributo border para definir o tamanho da borda da tabela.

Não existe um tag que crie colunas, apenas tags que criem linhas (<tr></tr>) e células (<td></td>). O ideal é que cada linha possua a mesma quantia de células para que não haja espaços em branco na tabela. Usamos a tag <th><th> para inserirmos títulos para a coluna da tabela.

A tabela possui duas áreas a cabeça e o corpo, a cabeça é representada pela tag <thead></thead>, onde colocamos os títulos da tabela (<th></th>); o corpo é representado pela tag <tbody></tbody>, onde inserimos as linhas e células da tabela.
