Formulários

Os formulários são ferramentas muito usadas nas páginas, usamos eles quando precisamos que o usuário nos passe alguma informação, seja escolhendo uma das opções pre-definidas, seja inserindo dados. Vale lembrar que com somente HTML e CSS não é possível receber as informações que o usuário enviar, para isso seria necessário o uso de uma linguagem de programação.

Criando um formulário

Iniciamos um formulário com a tag <form></form>. Vale lembrar que podem ser exibidos na tela do usuário mais de um formulário caso desejar. Essa tag pode receber o atributo "action" que vai dizer ao navegador para onde irão as informações inseridas no formulário, o valor inserido é o endereço para o qual as informações serão enviadas. Outro atributo que pode ser inserido nessa tag é o atributo "method" que especifica o método de envio das informações, esse atributo só recebe dois valores, GET e POST.
