Textarea

Outra tag que pode ser colocada em nosso formulário é a tag <textarea></textarea>, que como já indica o nome, criar uma área para inserção de texto.

Essa tag possui o atributo "row" que define quantos caracteres podem ser inserido em cada linha e o atributo "line" que define quantas linhas serão visíveis na área de texto. Porém o ideal é mudar a altura e largura da área de texto com CSS, já que o tamanho das letras pode variar de navegador para navegador.
