Inputs no HTML5

Aqui serão mostrados Inputs que são variações do atributo text (caixa de digitação) que foram adicionadas no HTML5, logo, não são suportadas por navegadores mais antigos (Caso sejam acessados esses inputs em um navegador antigo ele será mudado para input text). Sendo eles:

  • color: Cria um botão que quando clicado abre uma paleta de cores para o usuário escolher uma.
  • date: cria uma caixa de digitação pré-formatada para receber datas.
  • datetime-local: cria uma caixa de digitação pré-formatada para receber data e horário.
  • month: cria uma caixa de digitação pré-formatada para receber mês e ano.
  • number: cria uma caixa de digitação pré-formatada para receber apenas números
  • range: cria uma linha horizontal com uma bola que podemos mover pela linha.
  • search: cria uma caixa de digitação para pesquisa.
  • tel: cria uma caixa de digitação pré-formatada para receber um número de telefone.
  • time: cria uma caixa de digitação pré-formatada para receber o horário.
  • url: cria uma caixa de digitação pré-formatada para receber uma URL.
  • week: cria uma caixa de digitação pré-formatada para receber a data baseada em semanas.
