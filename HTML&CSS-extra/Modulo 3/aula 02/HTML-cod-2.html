Formatação de texto no CSS parte 2

Aqui serão mostradas outras formatações de texto com CSS que, embora não sejam as principais, são interessantes de usar.

  • text-transform: os principais valores colocados nesse atributo são:
    • uppercase: deixa todas as letras do texto dentro do elemento que recebe esse valor em maiúscula
    • lowercase:deixa todas as letras do texto dentro do elemento que recebe esse valor em minúscula
  • letter-spacing: define o espaçamento entre um letra e outra (em px por exemplo)
  • word-spacing: define o espaçamento entre uma palavra e outra (em px por exemplo)
  • line-height: define o espaçamento entre um linha e outra de texto (em px por exemplo)
  • text-shadow: usado para criar uma sombra em baixo do conteúdo dentro do elemento de recebe esse atributo
  • :

Exemplos:

        p {
        text-transform: uppercase;
        }
        p {
            text-transform: lowercase;
            }
        p {
            letter-spacing: 10px;
            }
        p {
            word-spacing: 20px;
            }
        p {
            line-height: 50px;
            }
        p {
            text-shadow: 1px 1px 
            1px red;
            }
