Fontes Externas

Usamos fontes externas quando queremos que o nosso site apareça com determinada fonte para todos que o acessarem, mesmo que seu computador não possua a fonte.
Para usarmos fontes externas devemos importar a fonte para o nosso site.

Adicionando uma fonte externa

Usando o google fonts, procuramos pela fonte que mais nos agrade, após escolhermos, clicamos no sinal de mais no canto superior direito da fonte, e clicar no +, após isso, vamos no canto superior direito do site onde diz "View Selected Famillies" e pegamos o link (nesse caso eu peguei o terceiro link) para a nossa fonte. Colocamos o link dentro da tag head. Após isso, ao atualizar o nosso site nada terá mudado, devemos especificar no CSS os elementos que vão receber esse font-familly (por ser uma fonte externa, devemos escrever o nome dela entre aspas).
