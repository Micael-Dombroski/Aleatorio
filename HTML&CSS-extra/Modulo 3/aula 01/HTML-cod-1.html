Formatação de textos com CSS parte 1

Para formatar determinado elemento com CSS devemos chamar ele pelo nome da tag, id, class ou pelo elemento pai (nesse último todos o elementos dentro do elemento pai serão afetados também).

Atributos para formatação de textos com CSS

  • color: muda a cor da fonte do texto
  • background-color: muda a cor de fundo do texto
  • font-size: muda o tamanho da fonte do texto (px, em, vh, vw, cm, etc)
  • font-weight:: recebe os valores "normal" e "bold", esse último é usado para deixar o texto em negrito
  • font-family: muda o tipo da fonte do texto. Ex: Arial, Times New Roman, Verdana, Serif, etc
    • Caso o navegador não tenha o fonte que colocamos como primeira opção, podemos colocar outras opção, ex: font-family: opção1, opção2, opção3,..., opção80;
    • Caso você só coloque uma opção de fonte e o navegador não possuir, ele usará a fonte padrão do navegador (geralmente é serif)
  • text,align: define o alinhamento do texto, os principais valores são: left (alinha o texto com o lado esquerdo do navegador), right (alinha o texto com o lado direito do navegador), center (alinha o texto com o centro do navegador), justify (justifica o texto)
  • text-decoration: define se o texto terá decoração e que tipo de decoração ele terá (none [sem decoração], underline [deixa uma linha embaixo do texto], overline [deixa uma linha em cima do texto], line-through [deixa o texto riscado], etc) etc)
