Select

Além do input, podemos também colocar a tag <select></select> dentro de um formulário.

O select cria uma caixa que, quando clicada, exibe as opções inseridas dentro da tag.

Para inserirmos as opções que o usuário pode selecionar na tag select, usamos a tag <option></option>

Para que seja exibido um valor predefinido na tag select, devemos inserir na option desejada o atributo "selected".

Para transformar a tag select em múltipla escolha devemos inserir o atributo "size" o valor inserido nele correspnderá a quantidade de itens que serão visualizados; e o atributo "multiple", que indica que o select é de múltipla escolha (para selecionar mais de 1 é necessário após selecionar 1 item, pressionar a tecla CTRL e selecionar os outros itens).
