Listas no HTML

Temos no HTML a tag <ul></ul> que faz listas desordenadas e a tag <ol></ol> que faz listas ordenadas, elas são as principais listas que usamos no nosso dia a dia.

Estilizando Listas

Podemos mudar os marcadores tanto das listas desordenadas quanto das listas ordenadas. Para isso, colocamos dentro desse elemento, no CSS, o atributo 'list-style-type' para a lista desordenada os seguintes valores: disc, square, circle, none;
e o atributo 'type' dentro da tag <ol></ol>, que recebe os seguintes valores: 1 , a, A, i, I

Definindo início de contagem das listas ordenadas

Para definirmos o número pelo qual a lista deve começar, devemos colocar dentro da tag <ol></ol> o atributo 'start', o valor inserido será o número pelo qual a listagem iniciará.
