Tag Input

Através dessa tag podemos captar um tipo de informação vinda do usuário, criando campos de entrada de dados no site. Essa tag não possui tag de fechamento (</>).

O principal atrbituto dessa tag é o atributo "type", onde será descrito o tipo de campo de entrada de dados será criada. O atributo "type" recebe os seguintes valores:

  • text: cria uma caixa de digitação, onde o usuário pode escrever dados.
    • password: cria uma caixa de digitação, porém os dados inseridos são visualizados como "*", esse campo é usado normalmente para o usuário inserir senhas.
    • email: cria uma caixa de digitação, onde o usuário pode inserir um endereço de email. É obrigatório inserir "@" no meio das informações inseridas (não pode ser o último e nem o primeiro dado inserido) dentro dessa caixa de digitação, caso o contrário o usuário é notificado sobre a ausência desse símbolo.
  • checkbox: cria um campo de marcação onde pode ou não estar ativado e o usuário pode ativar/desativar clicando nele.
  • radio: cria um campo de marcação onde pode ou não estar ativado e o usuário pode ativar/desativar clicando nele. Esse atributo é usado quando queremos dar várias opções para o usuário e ele só pode escolher uma, para isso, devemos colocar em todas os input radio do mesmo grupo o atributo "name" com o mesmo valor.
  • submit: envia todos os dados inseridos no formulário.

Atributo Name na Tag <input>

É importante colocarmos em todas as tags inputs o atributo "name" para que o navegador saiba que tipo de informação será inserida naquele campo.

Colocando valores predefinidos no nosso formulário

É possível colocar valores predefinidos no nosso formulário usando o atributo "value" dentro da tag <input> para caixas de digitação e para escrever algo dentro de um botão e o atributo "checked" quando for caixas de marcação (para deixar algumas marcada tanto para o tipo radio quanto para o tipo checkbox). Ex:

Escreva seu nome:
Escolha seu time: Azul Amarelo Verde Vermelho
Input atributo: text
Input atributo: email
Input atributo: password
Input atributo: checkbox
Input atributo: radio Sexo: Masculino Feminino
Input atributo: submit
