Principais formas de formatar texto

Para deixar o texto em negrito usamos a tag <strong>.

Para deixar o texto em itálico usamos a tag <en>.

Para deixar o texto menor que os demais de mesma font-size usamos a tag <small>.

Para indicarmos que uma informação do nosso texto não é mais válida usamos a tag <del>.

Para marcarmos um texto como se estivessemos usando marca-texto, usamos a tag <mark>.

Para colocarmos por exemplo um o flutuante ao lado de um número para indicar por exemplo primeiro, segundo e terceiro (1o, 2o e 3o) usamos a tag <sup>.

Para colocarmos por exemplo o número de determinado átomo presente num elemento de forma que ele apareça menor e mais abaixo da linha em que o texto está (Ex: H2O), usamos a tag <sub>.
