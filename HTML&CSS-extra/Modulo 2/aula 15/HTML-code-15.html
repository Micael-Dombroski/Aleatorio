Comentário HTML

Usamos os comentários para colocarmos observações para quem for observar o nosso código em HTML.

Para escrevermos um comentário basta fazermos isso: <!-- Comentário -->

Os comentários só são visíveis no código ou seja, não podemos ver eles na página.

Devemos colocar o comentário para explicar o que virá a seguir dele e não o que veio antes.
