O que é Identação

É uma forma de organizar o código, para mostrar por exemplo o que faz para de qual tag e o que não faz.

            

                <tag1>
                    <tag2>
                </tag1>
                <tag3>
                    <tag4>
                        <tag5>
                            <tag6>
                        </tag5>
                    </tag4>
                </tag3>

A identação serve na maioria das linguagens para organizar o código.

No VSCODE a tecla tab insere um espaçamento equivalente a 4 espaços por padrão. Em resumo, a identação serve para deixar o código com os espaçamentos corretos entre as tags e as linhas seguintes.
