Todo elemento que é visualizado na tela possui essas 3 propriedades, o conteúdo, a borda e o padding, o margin.

Margin

Usamos atributo "margin" para criar um distância entre o elemento e os demais elementos ou mesmo as bordas do navegador. Ex:

<p style="border: 2px solid; margin: 20px;">Parágrafo</p>

Podemos também aplicar o margin para somente uma das bordas especificando, desta forma: "margin-top".

Padding

Usamos o atributo "padding" para aumentar a distancia do contéudo interno do elemento e a borda dele. Ex:

<p style="border: 2px solid; padding: 20px;">Parágrafo</p>

Podemos também aplicar o padding para somente uma das bordas especificando, desta forma: "padding-top".

Especificando o margin ou padding de todos os lados quando usar valores diferentes

Para isso sempre seguimos essa ordem de inserção de valores: cima, direita, baixo, esquerda. Ex:

<p style="margin: 20px 10px 50px 5px;">Parágrafo</p>

Se colocarmos somente 2 valores, o primeiro valor será aplicado para cima e para baixo e o segundo para a direita e para a esquerda.

<p style="border: 2px solid; padding: 20px 50px;">Parágrafo</p>
