Links

Para fazermos uma âncora na nossa página, isto é, fazer uma ligação entre nossa página e outra página, usarmos a tag <a>, que permite que criemos links que quandos clicados, encaminham o usuário para o destino previamente definido dentro dessa tag.

A tag <a> tem o atributo "href" onde inserimos o link para onde o usuário será enviado após clicar no elemente dentro dessa tag.

Podemos criar links para outros sites, para arquivos e até mesmo para outras páginas do nosso próprio site.

Temos também outro atributo muito usado dentro dessa tag, o atributo "target", que é normalmente usado para que quando o usuário clique no link, ele seja aberto em outra aba ou mesmo janela do navegador. Para fazer isso, basta inserir o valor "_blank" dentro do atributo. Normalmente é usado esse atributo para inserir links externos em nossa página (isto é, link para páginas que não são do mesmo site que o nosso), pois faz com que o usuário mesmo clicando no link, não abandone a nossa página por completo.

Outro atributo bastante utilizado é o atributo "title", onde inserimos um título para o nosso link. Ao inserirmos um título no link, sempre que colocarmos o cursos do mouse em cima dele, irá aparecer um balão perto do link com o título dele escrito dentro. É usado geralmente para explicar resumidamente o que o usuário irá encontrar na página que o link irá encaminhar ele.
