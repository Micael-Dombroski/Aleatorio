Como definir a altura de um item com CSS

Para isso usamos no CSS o atributo height para o item.
Esse atributo pode receber o valor em %, cm (centímetros) e em px (pixels) por exemplo, também podemos usar o valor auto onde o item irá se ajudar automaticamente ao espaço disponível, sempre se expandido o máximo possível dentro do seu limite, o valor inherit também chamado de herança, faz com que o elemento pegue o mesmo valor de altura ao do item no qual ele está aninhado (dentro).

Podemos usar o atributo max-height para definir uma altura máxima que o item seja permitido se expandir, não podendo passar desse valor máximo, e um valor mínimo no qual o item é proíbido de ficar menor que esse valor com min-height.
