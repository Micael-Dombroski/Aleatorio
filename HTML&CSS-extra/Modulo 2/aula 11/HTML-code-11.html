Height & Width

Quando definimos a altura e/ou largura de um elemento, estamos definindo esses valores para o conteúdo, caso criemos uma caixa de 400px de largura e altura e adicionemos um padding de 10px e border de 5px a largura do quadrado no site será de 430 (10px na esquerda + 10px na esquerda + 5px na esquerda + 5 px na direita= 30px).
Abaixo um exemplo com 2 quadrados de 400px porém um com padding e border com os valores mostrados no texto acima.

Com isso notamos que a altura real do elemento é a soma do conteúdo interno, da margin e da borda dele.

Como definir uma altura/largura para o elemento que não será alterada pelo padding nem pelo border?

Escrevemos o seguinte código no CSS (preferencialmente no topo dele):
* {
box-sizing: content-box;
}
O valor que devemos usar é:
* {
box-sizing: border-box;
}

O quadrado do meio possui os mesmos atributos e valores que o primeiro quadrado no entanto, aplicamos no atributo box-sizing o valor border-box ao invés de deixar o valor padrão.

Importante

Devemos sempre colocar no estilo do nosso site o comando * {box-sizing:border-box;} e aplicar dentro do body margin:0px.
