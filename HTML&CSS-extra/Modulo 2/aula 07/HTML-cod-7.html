Cores no CSS

Podemos alterar a cor da fonte e fundo de diversos elementos HTML, como links, títulos, parágrafos, códigos, etc. Para isso, existem 3 formas de aplicar cores a elementos HTML no CSS, veja a seguir quais são.

Color name

Esse é o método mais fácil, nele escrevemos apenas o nome da cor que queremos em inglês, no entanto, esse método é limitado pois o CSS só reconhece o nome de cerca de 140 cores diferentes.

RGB

Por padrão em munitores e computadores as cores são formadas pela mistura de 3 cores em diferentes proporções, sendos estar Vermelho (Red), Verde (Green) e Azul (Blue), formando o que chamamos de Sistema de Cores RGB (Red, Green, Blue). A proporção de cada uma dessas cores varia de 0 a 255, caso todas estejam em 0 a cor formada será preto, caso todas estejam em 255 a cor formada será branco.

Motivo de 255 ser o valor máximo

Um código RGB é uma representação dos valores das cores primárias juntas, cada cor pode ser representada por um número de 0 a 255. (o número máximo é 255 pois por mais que haja 16 números possíveis e 16x16=256, quando são 2 zeros o valor é igual ao de somente 1 zero, então é desconciderado 1 desses zeros, ficando assim 256-1=255).

RGB Hexadecimal

Um código RGB é uma representação dos valores das cores primárias juntas, só que em Hexadecimal. Podemos perceber que os códigos RGB são sempre formados com 6 dígitos, o que dá 2 dígitos para cada cor. Se em 2 dígitos hexadecimais conseguimos valores de 00 a FF, então conseguimos representar valores de 0 a F(15) da base decimal.
