Formas de selecionar um elemento no CSS

Existem diferentes formas de selecionar um elemento no CSS, abaixo estão algumas delas:

  • Através do próprio nome ex: h1 {color: red; size: 20px;}

Nessa primeira forma, atribuimos a todos os elementos com essa tag os atributos, caso seja preciso atribuir a apenas um elemento determinado atributo, podemos usar esses 2 métodos:

Dando um id para o elemento desejado, ex:


        <html>
            <head>
                <style>
                    #titulo1 {
                        background-color: black;
                        color: white;
                    }
                </style>
            </head>
            <body>
                <h1 id="titulo1">Título 1</h1>
                <h1>Título 2</h1>
                <p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. 
                Quam et repudiandae quis obcaecati aut placeat fugiat, fugit perspiciatis odio blanditiis accusamus itaque dolorum, eveniet non commodi ab deleniti ratione unde.</p>
            </body>
        </html>

Ou dando uma classe para o elemento desejado, ex:


        <html>
            <head>
                <style>
                    .letra-branca {
                        background-color: black;
                        color: white;
                    }
                </style>
            </head>
            <body>
                <h1 class="letra-branca">Título 1</h1>
                <p class="letra-branca">Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia 
                et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
                <h1>Título 2</h1>
                <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia 
                et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
            </body>
        </html>

Quando usar class e id?

Podemos usar o id sempre que quisermos aplicar atributos a somente um elemento, já o class, usamos quando queremos aplicar o mesmo atributo para elementos de tags diferentes ou em apenas alguns elementos de tags iguais. Note que a forma de chamarmos um elemento por id e class são diferentes, para chamarmos um id no CSS devemos antes de escrever seu nome inserir "#" e para chamarmos um class devemos color "." antes do nome da class.

Como aplicar atributos para id e class simultâneamente?

Para isso fazemos o seguinte:


            <html>
                <head>
                    <style>
                        .letra-branca, #titulo-1 {
                            background-color: black;
                            color: white;
                        }
                    </style>
                </head>
                <body>
                    <h1 class="letra-branca">Título 1</h1>
                    <p class="letra-branca">Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia 
                    et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
                    <h1 id="titulo-1">Título 2</h1>
                    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia 
                    et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
                </body>
            </html>

Note que usamos vírgula para atribuir os atributos a mais de um item (no caso o item id titulo-1 e o item class letra-branca).

Como selecionar um elemento dentro de outro com CSS?

Para isso fazemos o seguinte:


        <html>
            <head>
                <style>
                   ol li {
                       background-color: red;
                       color: white;
                   } 
                </style>
            </head>
            <body>
                <ol>
                    <li>Item 1</li>
                </ol>
            </body>
        </html>

Note que indicamos no CSS que queremos mudar os atributos do elemento presente na tag <li></li> dentro da tag <ol></ol>
