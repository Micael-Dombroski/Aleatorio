O que é e como usar o CSS?

O CSS é responsável por alterar e inserir novas propriedades nos elementos presentes no documento HTML (aumentar o tamanho do elemento, mudar a cor da fonte do texto, etc).

Enquanto o HTML é responsável por fornecer a estrutura para a página web, o CSS é responsável por estilizar a página, deixando ela mais apresentável para os usuários.

Como usar o CSS

Existem 3 formas de inserirmos o CSS no HTML:

  • CSS inline. Ex: <p style="color: aqua;"></p>
  • CSS interno. Ex: 
                     
                    <head>
                    <style>
                        body {
                            font-family: 'Times New Roman';
                        }
                    </style>
                    </head>
  • CSS externo Ex: 
    
                <head>
                <link rel="style sheet" href="Link-Externo-CSS.css">
                </head>
