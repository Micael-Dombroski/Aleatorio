Links

Podemos fazer links com imagens, gifs, inúmeros elementos que podemos inserir dentro da tag <a>, e todos os elementos dentro dela fazem parte do links, caso tenha texto e imagem nele, ambos encaminharam o usuário para o destino previamente determinado no link quando clicado.

Podemos usar os links também para nos levarmos para áreas específicas dentro de nossa página, sem encaminhar o usuário para outra página. Exemplo:

  • Clique aqui para ver a Justice
  • Clique aqui para ver a Lucifer
  • Clique aqui para ver a Malina
  • Clique aqui para ver a Modeus
  • Clique aqui para ver a Loremaster
  • Clique aqui para ver a Beelzebub

Estilizando Links

Para estilizarmos links do jeito comum é só escrevermos os estilos desejados dentro do CSS do arquivo. Ex:
<style>
a {
color: green;
font-size: 30px;
}
</style>

Para estilizarmos como o link ficará quando o mouse estiver em cima dele devemos escrever no CSS da seguinte forma:
<style>
a:hover {
color: pink;
font-size: 20px;
}
</style>

Para estilizarmos como o link que já foi visitado aparecerá devemos escrever no CSS da seguinte forma:
<style>
a:visited {
color: red;
font-size: 40px;
}
</style>
