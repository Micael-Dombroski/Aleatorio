Tags Básicas

O que são tags Básicas?

São as tags que vamos usar normalmente dentro do HTML. Abaixo vamos mostrar essas tags:

  • <h(1,2,3,4,5,6)>Título</h>
  • <p>Parágrafo</p>
  • <a href="www.google.com">Clique Aqui</a>
  • <img src="link-da-imagem" alt="texto-alternativo-para-a-imagem">
  • <button>Clique em mim</button>
  • <br>
  • <ul></ul>
  • <li>Item 1 Da Lista</li>
  • <ol></ol>
