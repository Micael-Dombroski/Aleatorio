CSS interno

Nesse modelo, o CSS é inserido dentro do documento HTML, em uma parte só, diferente do inline onde temos CSS em espalhado em cada tag. Colocamos o CSS dentro do documento HTML através da tag <style></style> dentro da área head, é recomendado colocar essa tag como último item dentro da tag <head></head> (se não me engano). 


        <head>
        <style>
            body {
                background-color: black;
                text-align: center;
            }
            p, h1 {
                color: white;
                font-family: 'Times New Roman'
            }
        </style>
        </head>

CSS externo

Nesse modelo, o CSS está em um arquivo separado de extensão .css, que é linkado ao documento HTML dentro da área head do documento. Essa é a maneira mais organizada de trabalhar com o CSS. 


        <head>
        <link rel="stylesheet" href="nome-do-doc.css">
        </head>
