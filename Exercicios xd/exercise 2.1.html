Quebrando linhas

Usando a tag <br> podemos fazer a quebra de linha em um parágrafo.

Mesmo Usando a tecla "espaço" para separar as linhas pelo arquivo, na página elas apareceram numa linha consecutiva igual a um parágrafo.

Elas podem aparecer em linhas diferentes se escrevermos as palavras em parágrafos diferentes abrindo mais de uma tag de parágrafo ou usando a tag <br>

Para inserir emojis na nossa página devemos pegar seu código na emojipedia

Escolherei esse emoji ;), na emojipedia ele possuí o codepoint U+1F609 porém esse código não pode ser usado para inserir o emoji, então precisamos adaptar o código
substituindo o "U+" por "&#x", fazendo isso, conseguimos inserir o emoji 

Além disso, temos alguns sinais como o de "menor que" e "maior que" para abrir tags no site, devemos inserir um código para representar esses símbolos
esses códigos são chamados de HTML Entites e podemos acessar o site da W3School na área de Misc Symbols para visualizar os códigos usados para representar esses símbolos.
