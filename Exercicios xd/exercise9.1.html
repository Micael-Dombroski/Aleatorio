Tipos de Links

Existem 2 tipos de links, o externo e o interno.

Links Externo

Os links externos são links que quando clicado levam o usuário para outro site. Este links vai te direcionar a outro site Porém esse link acima vai substituir o conteúdo que o usuário está vendo pelo conteúdo da página linkada. Mas podemos controlar onde o site de destino do link vai abrir usando atributos dentro da tag Anchor.

Como controlar onde o site de destino do link vai abrir

Podemos usar o atributo target para controlar onde o site de destino do link vai abrir. Este atributo possui os seguintes valores:

  • _blank

Natureza do destino do link

Com o atributo rel podemos indicar qual é a natureza do destino do link usado.
Esse atributo pode receber os seguintes valores:

  • next
  • prev
  • author
  • external
  • nofollow

Link Interno

Existem 2 tipos de links internos, em um dos tipos é feito a relação com outra página que está dentro do nosso servidor e condiz com nossa página. Ex: Para acessar a próxima página do nosso site clique aqui.

Links para Downloads

Para fazermos links para downloads basta fazer o link diretamente para o arquivo que deseja efetuar o dowload e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado. Temos todos o types para cada tipo de documento no site iana.org media types. Os valores mais usados são:

  • application/zip
  • application/pdf
  • text/html
  • text/css
  • text/javascript
  • video/mp4
  • video/H264
  • video/JPEG
  • audio/aac
  • audio/mpeg
  • font/ttf
  • image/jpeg
  • image/png

Lembrando que cada um dos tipos acima só deve ser usado se o documento for na formatação que é descrita em cada um dos itens da lista.
