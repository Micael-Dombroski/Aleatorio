CSS

O que é CSS?

CSS é uma sigla para Cascading Style Sheets isto é, folhas de estilo em cascata.

Qual a utilidade do CSS?

O HTML ele é responsável por inserir o conteúdo na nossa página, no entanto, se esse conteúdo não estiver bem organizado, efeitado, de forma que vá puxar a atenção do usuário para ela, ela poderá perder muito de seu potencial. O CSS é responsável por fazer essa estilização da nossa página.

Formas de aplicar o CSS ao nosso documento HTML

Existem 3 formas de aplicarmos CSS a nossa página HTML, podemos aplicar o CSS inlie style, CSS internal style e CSS external style. Iremos abordar essas formas logo a seguir.

CSS inline style

É a forma mais simples de aplicar estilos.
Nesse estilo o CSS é inserido dentro da tag do elemento que queremos mudar os atributos. Ex:

<p style="font-family: Arial; color: blue;">Lorem ipsum dolor sit amet consectetur, adipisicing elit. Illo officiis amet, esse ullam veniam vero odio at quaerat excepturi facere tempore error. Consequatur, illum commodi libero sit aliquid repellendus laborum?</p>

Note que o CSS foi inserido na tag <p></p> através do atributo style.
Esse método não é muito usad devido a poluição do código já que é preciso colocar o atributo style dentro de cada tag que desejamos mudar os atributos, é recomendado usar ela apenas para mudar os atributos de poquíssimas tags ao invés de todas as tags da página.

CSS internal style

É a forma mais interessante de aplicar estilos.
Para usarmos essa forma de inserirmos CSS na nossa página, devemos inserir dentro da tag <head></head> a tag <style></style>, e é nessa tag onde especificaremos o elemento presente no documento HTML ao qual queremos atribuir novas formatações e as formatações que queremos aplicar. Veja esse exemplo:


        <html>
            <head>
                <style>
                    h1 {
                        background-color: black;
                        color: white;
                    }
                    p {
                        background-color: blue;
                        color: red;
                    }
                </style>
            </head>
            <body>
            

                
               <h1>Título 1</h1>

                    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia
                et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
                
               <h1>Título 2</h1>

                    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia
                et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
            

            </body>
        </html>

CSS external style

Nessa forma de aplicar o CSS criamos um arquivo externo que irá abrigar todo o código CSS (o arquivo deve possuir extensão .css), essa forma de usar CSS é muito útil quando queremos aplicar o mesmo estilo a mais de uma página (muito usado para estilizar páginas que são extensões da página principal). O lado bom dessa forma de usar CSS é que precisamos criar somente um arquivo .css para estilizar inúmeras páginas que possuiram o mesmo estilo.
Conectamos o arquivo .css ao arquivo html inserindo um link dentro da área <head></head>. 


        <html>
            <head>
                <link rel="stylesheet" href="style.css"> 
            </head>
            <body>
            

                
               <h1>Título 1</h1>

                    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia
                et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
                
               <h1>Título 2</h1>

                    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Dolor laborum, facilis rem, eius aperiam quia
                et molestiae numquam, quidem fugit impedit sit at explicabo itaque beatae quam expedita aliquid vitae.</p>
            

            </body>
        </html>

Ordem de preferência entre as formas de aplicar o CSS

Existe uma ordem de preferência de qual estilo aplicar se o documento HTML possuir as 3 formas de inserção de CSS nela. Está é a ordem:

  1. CSS Inline Style
  2. CSS Internal Style
  3. CSS External Style

Sempre o estilo aplicado de forma Inline irá se sobrepor diante os outros e sempre o estilo External será o último a ser aplicado e só será aplicado em tags que não são afetadas pelo CSS Inline e Internal.
