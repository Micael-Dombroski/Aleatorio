Criando Formulários

Usamos a tag <form></form> para criar um campo na página que vai abrigar todo o formulário. Os campos de entrada onde o usuário vai inserir dados, usamos a tag <input>, podemos usar diferentes valores no atributo type que podem mudar o tipo de informação que o usuário poderá nos enviar através do formulário.

Devemos colocar sempre na tag <input> o atributo name pois ele define o nome da variável que receberá o conteúdo do campo. É essencial para a criação de qualquer campo de entrada de dados;

Usamos o atributo placeholder para deixar um texto dentro da caixa de digitação por padrão, ajuda a guiar o usuário.

Valores usados no atributo type

  • text: cria um campo de digitação onde o usuário pode escrever coisas. Use o atributo maxlength para especificar o comprimento máximo do valor que pode ser inserido e size para especificar a quantidade de digitor que podem ser visualizados no campo;
  • radio: cria uma caixa de verificação, geralmente usamos ela para dar mais de uma opção para o usuário porém só podendo selecionar uma. Usar o atributo value para definir o valor a ser enviado por este item. Use o atributo checked para indicar se este item deve estar selecionado por padrão. Botões de escolha que têm o mesmo valor para o atributo name estão no mesmo "grupo de botões de escolha"; apenas um botão de escolha no grupo pode estar selecionado de cada vez;
  • checkbox: cria uma caixa de verificação, geralmente usada para dar várias opções ao usuário e permitir que ele selecione mais de uma delas;
  • submit: cria um botão que quando clicado envia o formulário (caso tenha um destino para onde o formulário possa ser enviado);
  • reset: cria uma botão que quando clicado reseta todos os dados inseridos no formulário para o dados iniciais dele;
  • data: cria um campo de digitação para o usuário inserir dia, mês e ano;
  • datatime-local: cria uma campo de digitação para inserir data e horário, sem fuso horário;
  • email: cria uma campo de digitação para inserir o email;
  • password: cria um campo de texto com uma só linha cujo valor é obscurecido. Use o atributo maxlength para especificar o comprimento máximo do valor que pode ser inserido (server para TEXT também) e size para especificar a quantidade de digitor que podem ser visualizados no campo (serve para TEXT também);
  • number: cria um campo de texto onde podem ser escritos somente números inteiros;
  • range: cria uma linha com uma bola onde podemos mover a bola para a direita para aumentar o valor ou para a esquerda para diminuir. Por padrão os valores dos atributos min e max são 0 e 100 espectivamente;
  • search: cria um campo de texto para pesquisa de termos de busca;
  • hidden: define um campo invisível, cujo o conteúdo é enviado junto com os demais;
  • tel: cria um campo de texto para inserir o número de telefone;
  • time: cria um campo de texto para inserir o horário;
  • file: Permite ao usuário selecionar um arquivo do seu computador. Use o atributo accept para definir os tipos de arquivo (.png, .jpg, etc) que serão exibidos na procura;
  • week: cria um campo onde podemos inserir a semana do ano (exemplo: semana 20 de 2022);
