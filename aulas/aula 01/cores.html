Estudo das cores para as páginas

A escolha das cores que serão usadas para nossa página implica na apresentação dela, se escolhido bem a paleta de cores, tudo ficará mais agradável visualmente para o leitor. Fazendo com que ele permaneça por mais tempo no seu site e até mesmo sendo decisivo para a efetivação de uma compra no mesmo.

A emoção das cores

Segundo Neil Patel as pessoas levam cerca de 90 segundos para decidir se querem ou não um produto, e que 90% dessa decisão se baseia na sua cor.

As cores nos transmitem emoção mesmo que de isso não ocorra na maioria das vezes de forma totalmente consciente. Sentimos a emoção das cores de um site ou produto mesmo sem saber que quem o modelou usou a pscologia das cores para modelá-lo.

Se pesquisarmos sem se aprofundar muito no tema da "psicologia das cores", já será possível observar várias sugestões de emoções voltadas para determinada cor. Um exemplo é a cor azul, que remete a harmonia, equilíbrio, confiança, profissionalismo, integridade e segurança. A cor azul é usada em vários logotipos como por exemplo: Facebook, Twitter, LinkedIn, Dell, HP e Intel. Claro que isso não é por acaso.

Harmonia das cores

Sempre devemos usar cores que fassam sentido juntas, para que quando o usuário visualize o site sinta aquela sensação de que tudo está em perfeito harmônia, fazendo-o parecer ainda mais belo. Por isso é importante decidirmos com cuidado qual será a paleta de cores usada na construção da página.

Cículo Cromático

O círculo cromático é a base para saber escolher cores que vão gerar um sentido harmônico para o nosso site. Abaixo, uma imagem de um círculo cromático:

Podemos ver que no círculo cromático temos as três cores primárias, da junção das cores primárias temos as três cores secundárias e da junção de uma cor primária e uma secundária, temos as seis cores terciárias.

Além da temperatura, podemos classificar as cores por esquemas harmônicas.

Esquemas Harmonicos

Cores complementares

São aquelas que apresentam maior contraste entre si. Elas estão localizadas do lado imediatamente oposto do círculo cromático. A cor complementar de uma cor primária sempre será uma cor secundária. A cor complementar de uma cor terciárias sempre será outra cor terciárias, sempre que juntamos duas cores complementares, obtemos a cor cinza.

Cores Análogas

As cores análogas são cores que são vizinhas entre si. Por serem consecutivas elas possuem um baixo contraste entre elas, mas criam uma bela harmonia quando combinadas em um mesmo design.

Cores análogas mais uma complementar

Juntansdo os dois tipos anteriores, essa técnica quebra um pouco o ritmo semelhante das cores análogas, adicionando uma cor que criar um grande contraste com as três análogas.

Cores análogas relacionadas

Nesse tipo de harmonia, é escolhido duas cores análogas (consecutivas) e depois pulamos por uma terceira cor (em qualquer direção) e escolhemos a quarta. Essa técnica proporciona um pouco mais de contraste que a técnica das análogas sem precisar usar uma cor complementar.

Cores intercaladas

É uma técnica de harmonia usada com menor frequência, já que nem sempre funciona como o esperado. Nessa técnica escolhemos uma cor e as duas seguintes escolhemos com intervalos contantes entre elas ( pulando a mesma quantidade de cores da cor 1 para a 2 e da 2 para a 3 ).

Cores triádiacas

Essa técnica garante uma grande riqueza de cores, onde escolhemos três cores equidistantes (distância igual entre todos os 3 pontos) no círculo cromático. Esse esquema gera sempre um triângulo equilátero e cria uma opção que sempre possui um ótimo contraste entre as cores.

Cores em quadrado

Selecionamos quatro cores equidistantes. Essa técnica permite um contraste mais rozoável entre as cores escolhidas. Esse esquema sempre gera um quadrado perfeito com os pontos selecionados.

Cores tetrádicas

Com essa técnica vamos escolher dois pares de cores complementares, que não serão necessariamente análogas (consecutivas). Isso garante dois pares de cores com bastante contraste entre si.

Monocromia

Diferente das harmonias anteriores, essa usa apenas uma cor e varia apenas a sua saturação e seu brilho. Essa técnicagera pouco contraste entre as cores escolhidas mas gera um visual agradável aos olhos, conhecido como "degradê".

Aplicando CORES ao nosso site

No CSS, ao criar uma propriedade relacionada a cores, se posicionarmos o cursos entre os dois pontos e o ponto de vírgula da declaração e darmos um clique e em sequência pressionar Ctrl + Espaço veremos todas as cores possíveis para serem escolhidas no CSS através do nome.

Porém esse método é muito limitado, pois a tela moderna é capaz de exibir cerca de 65 milhões de cores.

Para conseguirmos mais possibilidades devemos recorrer aos códigos hexadecimais ou então às funções CSS rgb ( ), rgba ( ), hsl ( ) ou hsla. Para usar eese recurso, adicione qualquer vor textual e passe o mouse sobre o nome da cor (veja a imagem abaixo) e uma janela especial aparecerá.

Como representar uma cor?

Todas as cores em uma tela são compostas da junção de três cores primárias: vermelho (red) , verde (green) e azul (blue).
Analisando a imagem abaixo, vemos que a junção das cores primárias nos leva a outras cores como o magenta, amarelo e ciano. Se usamos todas as cores primárias no máximo, chegamos ao branco.

Usando Gradientes em CSS

Podemos gerar gradientes e aplicarmos a componentes visuais usando folhas de estilo. Vamos usar um exemplo simples. Abaixo foi criado uma div que receber a seguinte declaração no CSS: 

    <style>
        body {
            font-family: Arial;
            background-image: linear-gradient(90deg, yellow, red);
            color: black;
            text-align: center;
            border: 1px solid black;
            padding: 20px;
        }
    </style>
Exemplo

Gradientes radiais

Podemos criar um gradiente em formato radial círcula, olhe o exemplo abaixo:

    <style>
        body {
            font-family: Arial;
            background-image: radial-gradient(red 10%, yellow 40%, green 50%);
            color: black;
            text-align: center;
            border: 1px solid black;
            padding: 10px;
        }
    </style>
