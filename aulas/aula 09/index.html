Anatomia do Tipo

Tipo é cada uma letra do alfabeto, aqui, será analisado as partes de cada letra do alfabeto, como elas serão desenhadas. Quando uma fonte é desenvolvida é preciso saber as medidas das letras nessa fonte, é importante saber as medidas para saber se elas se adequam aos tamanhos disponíveis. A altura do corpo da fonte é a altura total da letra mais alta dessa fonte.

Fontes serifadas facilitam a leitura da pessoa em papel. Glifos, também conhecidos como caracteres ou letra, quando em conjuntos com vários outros glifos de mesma família, criam uma fonte.

Família Tipográfica

Algumas fontes possuem variações, como por exemplo, a fonte open sans, que possui as seguintes variações: Light, Normal, Semi-bold, Bold, Extra-bold, cada uma dessas é uma família de fonte open sans.

Tipos de Fonte

Existem várias formas diferentes de escrever as letras, essas letras por sua vez, são encaixadas em uma fonte com letra que possuam o mesmo padrão, e para facilitar, temos os tipos de fonte, que nos ajuda na hora de procurar uma fonte para usar no nosso site. Dentre os tipos de fonte, temos: Fontes Serifadas (fontes que possuem seria, que são pequenos traços e prolongamentos nas extremidades das letras); Sans-serif (fontes que não possuem serifa); Monoespaçada (podem ter serifa ou não, mas todas as letras e símbolo possuem a mesma largura), Handwriting (fontes que copiam a escrita feita a mão) e por fim, as fontes Display (fontes que não se encaixam em nenhum dos tipos acima).

Fontes Serifadas X Fontes Sans-serif

Afinal, quando devemos usar fontes serifadas ou não serifadas? Bom, recomenda-se que para textos muito longos que serão exibidos em telas o uso de fontes Sans-serif, já para colocar em papel, é recomendado o uso de fontes serifadas. Devemos também colocar cores complementares, para textos muito longos devemos sempre usar texto preto e fundo branco, pois assim não cansa os olhos do visitante.
