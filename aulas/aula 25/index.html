Grouping Tags

É importante saber como agrupar as tags dentro de caixas em seu site para que ele fique bem organizado.

Antigamente as únicas tags de agrupamento eram a tag <div> que é do tipo box-level, e a tag <span> do tipo inline-level. Embora atualmente essas tags ainda possam ser usadas, elas não possuem um significado semântico, que é muito buscado no HTML5, por isso elas não são muito usadas em sites profissionais, ao invés delas, usamos outras tags de agrupamente que possuem significado semântico. Veja a seguir algumas delas:

Tags Semânticas do Tipo Box-level

  1. <header>: tag usada para marcar o cabeçalho da página.
  2. <nav>: tag usada para marcar aonde será o menu da sua página.
  3. <main>: tag usada para indicar aonde ficará o conteúdo da página.
  4. <footer>: tag usada para indicar aonde ficará o rodapé da página.
  5. <section>: tag usada para indicar seções na nossa página.
  6. <article>: tag usada para indicar artigos na página.
  7. <aside>: tag usada para colocar conteúdo periférico em relação ao artigo, por exemplo, da página, como por exemplo o nome do autor do artigo.
