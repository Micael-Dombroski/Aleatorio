Fontes Externas

Diferente das fontes que são exportadas para o navegador, as fontes externas estão baixadas no servidor do site, quando o site é aberto, o servidor envia uma cópia do arquivo da fontes para o navegador, que por sua vez vai deixar os elementos desejados com a fonte (somente os indicados no HTML/CSS).

Como aplicar fontes externas no CSS?

Usamos a propriedade @font-face na primeira linha do nosso CSS/ <style>, essa propriedade exige que seja inserido dentro dela o nome da fonte Externaonde ela es URLlocalizada dentro do servidor em relação ao arquivo HTML/CSS.

Ex de inserção de fonte:

    <style>
        @font-face {
            font-family: 'Love';
            src: url('love lerry.otf') format('opentype'), 
            url ('love-larry.ttf') format('truetype'); 
            font-weight: normal;
            font-style: normal;
        }
        h1 {
            font-family: 'Love', Times, serif;
            font-size: 3em;
            font-weight: normal;
        }
    </style>

Um site muito usado para pegar fontes externas é o DaFont, o Google Fonts também pode ser usado para isso, vale lembrar que sempre antes de baixarmos uma fonte externa para o nosso site devemos ver se ela suporta ou não caracteres especiais, além disso, algumas fontes só possuem os caracteres em maiúsculo ou em minúsculo.

Dica

É recomendado que as fontes externas do nosso site fiquem armazenadas em uma pasta separada dentro do local no servido onde a página é armazenada.
