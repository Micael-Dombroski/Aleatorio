Ferramentas Para Gerar uma Paleta de Cores

É importante sabermos ferramentas que podemos usar para gerar paletas de cores para usarmos em nossos sites. Abaixo estão listadas algumas ferramentas gratuitas:

  • Adobe Color (Se não for assinante não é possível salvar as paletas): Nessa ferramenta podemos escolhes o tipo de Técnica de armonização queremos usar para gerar uma paleta de cores e escolher o formato usado para gerar cores (rgb, hsl, etc), gera também o código hexadecimal de cada cor presente na paleta gerada. Pode extrair uma paleta de cor de uma imagem/logo;
  • Paletton: Nesse site também podemos escolher o tipo de técnica usada para gerar a paleta de cor do nosso site, porém, podemos ver um exemplo de como ficaria um site com a paleta de cor que foi gerada;
  • Coolors: Essa ferramenta é ótima quando não se tem uma cor principal já pré-definida, nesse site, ele vai sugerir paletas de cores que vão mudando conforme você vai pressionando a tecla Space do teclado. Você pode travar cores da paleta que você tenha gostado para que elas não mudem conforme você for apertando espaço, então o site vai procurar cores que combinem com as cores "trancadas".
