Usando Fontes Externas

O VS Code possui opções limitadas de fontes para usarmos em nossos sites, para usarmos fontes que não estão no VS Code devemos exportar elas de algum site de fonte, um exemplo de site é o Google Fonts, onde podemos filtrar as fontes pelo tipo e ver como as letras ficam quando escritas naquela fonte.

Google Fonts

Para usar as fontes do Google Fonts, podemos baixar elas ou, copiar o link do código da fonte para HTML (<link>) ou importar o código para CSS (@import).

Se voc}e copiar o código para HTML, você deve inserir ele na área <head> do seu HTML. Se você importar o código para o CSS, você deve colocar ele no topo da sua CSS.
