Tipos de Dados e Variáveis

Podemos escrever comentários nos nossos código em JavaScript, existem duas formas de fazermos isso, a primeira é colocando "//" antes de escrever o comentário (//Comentário), usamos essa forma de escrever comentários para escrevermos uma única linha de comentário; a segunda é colocando o comentário entre "/*" e "*/" (/*Comentário*/), usamos essa forma de escrever comentários para escrever mais de uma linha de comentário.
Abaixo mostrarei alguns exemplos de como e quando usar esses tipos de comentários: 

        
        <script>
            window.alert('Hello Everyone')
            window.confirm('Do you wanna have a MAD time?') //Janela com botão (Ok e Cancelar)
            window.prompt('Qual é seu nome?') //Vai perguntar ao usuário seu nome
        </scrip>

O que é uma Variável?

A memória do computador possui espaços vazio que podem ser ocupados por dados. Se colocarmos dados de forma desordenada vamos acabar perdendo dados, para que isso não ocorra devemos organizar o armazenamento de dados na memória do computador com variáveis. Para crairmos uma variável inserimos a palvra "var" no nosso código, para identificar a variável usamos identificadores que são nada mais que nomes que irão representar determinada variável. Podemos atribuir valores para as variáveis usando o sinal "=" (que em JavaScript significa "recebe") entre o nome da variável e o valor que será atribuído a ela. var indentificador=valor . Abaixo um exemplo:

var Caixa_de_Som = CD_dos_Baroes_da_Pisadinha
Podemos ler esse código da seguinte forma: "A variável Caixa_de_Som recebe CD_dos_Baroes_da_Pisadinha"

Ao inserirmos um valor para a variável ele será atribuído a ela dentro da memória do computador.

Existem variáveis de tamanhos diferentes.

Identificadores

Os identificadores são os nomes que damos para as variáveis, para atribuirmos um nome para uma variável devemos seguir algumas regras:

  • O nome do identificador pode iniciar com uma letra, com $ ou com _ (Ex: Churros; $ucesso; _Agaragã;etc);
  • O nome do identificador não pode começar com um número (Ex: 3squeleto;etc);
  • O nome do identificador pode conter letras e números (Ex: P4r4l3l3p1p3d0;etc);
  • O nome do identificador pode conter acentos ou símbolos (Ex: Média; Saúde&Sucesso; etc);
  • O nome do identificador não pode conter espaços em branco, no lugar deles usamos _ (Ex: Olá_eu_sou_o_Goku; etc);
  • O nome do identificador não pode ser uma palavra reservada (Ex: Palavras usadas para escrever comandos, como a palavra var; etc).

Dicas para dar nomes para variáveis:

  • Letras maiúsculas e minúsculas fazem diferença, se você dar o nome de uma variável usando letras maiúsculas, você deve escrevar o nome dessa variável sempre em maiúsculo, se só uma letra estiver em maiúsculo, essa letra deve ser colocada em maiúsculo sempre;
  • Devemos escolher nomes coerentes para as variáveis, não faz sentido eu colocar o nome Batata-Doce para uma variável que represente um salário;
  • Evite se tornar um 'programador alfabeto', aquele que da o nome das variável conforme o alfabeto (variável1 nome: A, variável2 nome: B, etc), ou um 'programador contador', aquele que todas variáveis estão em ordem numérica (n1,n2,n3);

Tipos de Dados

Existem diferentes tipos de dados, sendo estes: number, 'string', boolean.

Esses 3 são os tipos primitivos primordiais na liguagem de JavaScript.

Existem outros tipos primitivos, como por exemplo, o tipo primitivo "number" possui dois valores internos importantes, o "Infinity" e o "NaN".
Também existem os tipos primitivos "null", "undefined", "object" (já que o JS é uma linguagem direcionada a objetos, o null é considerado object), existem o tipo "array" que é um tipo primitivo interno do object, o tipo primitivo "function" já que ela também funciona no paradigma funcional.

Para trabalharmos com esse tipos todos existe um comando no JS chamado typeof

Podemos mudar o tipo de uma variável, veja o exemplo abaixo:

    
<script>
    var n = 354
    typeof n        (Resultado visualizado: number)
    var n = "Suco de maracujá"
    typeof n        (Resultado visualizado: string)
    
    //Note que mudamos o valor da variável n e o seu tipo, antes seu valor era 354 e seu tipo era
     number, depois passamos seu valor para "Suco de maracujá" e seu tipo mudou para string.
    
</scrip>
