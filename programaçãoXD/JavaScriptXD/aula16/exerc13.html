Function

Funções são um conjunto de passos que o computador deve seguir para realizar uma tarefa. Vamos ver abaixo um exemplo.

Ex: Sua mãe te chama, te entrega dinheiro e pede para você comprar um leite na mercearia.

  1. Assim como sua mãe te chama para pedir que você compre o leite, para uma função ser executada ela precisa ser chamada. Nem sempre as funções precisam ser chamadas para serem executadas, as vezes elas são executadas devido a algum evento, como por exemplo: recolher a roupa quando esta chovento, onde nesse caso, a função recolher a roupa é executado quando o evento chover ocorre.
  2. Assim como você precisa do dinheiro para comprar o leite, a função precisa de parâmetros que devem ser atendidos para a função ser executada, nesse caso os parâmetros são a ordem de sua mãe e o dinheiro.
  3. O processo de ir até a mercearia e comprar o leite é o que chamamos dentro de uma função como ação.
  4. Da mesma forma que ao retornar para casa você entrega o leite para sua mãe, a função também deve retornar algo e isso é chamado de retorno.

Vale lembrar que não são todas as funções que usam parâmetros e que dão algum retorno durante ou ao término de sua execução

Definição de Função na Programação

Funções são ações executadas assim que são chamadas ou em decorrência de algum evento.

Uma função pode receber parâmetros e retornar um resultado.

Vamos dar um exemplo em JS.


1        function parimp(n) {
2            if (n%2==0) {
3                return 'par'
4            } else {
5                return 'impar'
6            }
7        }
8        let res = parimp(11)

Nesse exemplo, na linha 8 temos a chamada para a nossa função, e na mesma linha já é determinado o parâmetro para a nossa função, na linha 1 temos a função, na linha 2 temos uma condição que averigua se o parâmetro fornecido na linha 8 quando dividido por 2 terá o resto da divisão como 0, caso isso esteja correto será retornado o valor 'par' presente na linha 3, caso esteja errado, será retornado o valor 'impar' que está presente na linha 5. Com isso podemos determinar se o número é par ou impar.

Também é possível usar uma função como valor de variável (olhe o exemplo no arquivo JS dessapasta).
