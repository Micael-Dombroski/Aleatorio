Relembrando alguns assuntos...

DOM é o Document Object Model, ele é um facilitador que existe dentro do seu navegador para dar acesso aos elementos diretamente pelo JS.

A Árvore DOM é mostra todos os elementos presentes na página, desde o que abriga todos até último abrigado.

Podemos chamar elementos do HTML no JS usando as seguintes técnicas: TagName (pelo nome da tag), ID (pelo ID colocado no elemento), Name (pelo nome colocado no elemento), ClassName (pela classe colocado no elemento) e querySelector (faz a seleção através de elementos CSS).

DOM Events

Eventos são tudo o que pode acontecer com determinado elemento. Ex: Avisar que o cursos do mouse está dentro de uma div (mouseneter), caso o mouse seja movido dentro de uma div avisar que o mouse está se movendo dentro da div (mousemove), caso seja mantido o botão de clicar dentro da div avisar que você estar pressionando o clique na div (mousedown), avisar quando você parar de pressionar o botão de clicar do mouse sobre a div (mouseup), avisar quando você der um clique rápido na div (click), avisar quando você remover o cursor do mouse de dentro da div (mouseout). Existem muitos outros eventos possíveis. Podemos ver outros eventos possíveis em outros elementos pesquisando "JS DOM event referent" e entrar no site da mozzila, ou também, você pode ser encaminhado para o site direto, Clicando Aqui

Funções

É um conjunto de linhas que só vão ser executadas quando determinado evento ocorrer, esse conjunto de linhas é chamado de bloco, que é colocado entre "{}". Devemos nomear o bloco de function, porém só nomeando assim, será criado uma função anônima, devemos complementar com o nome da ação executada pelo bloco, também podemos complementar com alguns parâmetros entre "()". Ex:
function ação (parâmetros) {
bloco
}

No exemplo criamos funções em HTML para a div, olhe o código para mais detalhes.
