Operadores

A linguem de programação JavaScript possui vários operadores, hoje iremos abordar os operadores aritméticos e os operadores de atribuição.

Operadores Aritméticos

São os operadores responsáveis por operações aritméticas

Vamos apresentar a seguir operadores binários (que precisam de dois operandos):

  • 5 + 2 = 7
  • 5 - 2 = 3
  • 5 * 2 = 10
  • 5 / 2 = 2.5
  • 5 % 2 = 1
  • 5 ** 2 = 25

Precedência de operadores

Existe uma ordem de prioridade de operações as serem realizadas, essa ordem vai depender de seu operador. A seguir uma lista com essa ordem de precedência:

  1. ()
  2. **
  3. /,%,*
  4. +,-

Observação: Quando os operadores possuem a mesma ordem de precedência eles são resolvidos da esquerda para a direita.

Operadores de Atribuição

Usamos esses operadores para guardar dados, como por exemplo o valor de variáveis.
Exemplos de atribuições:
Atribuição Simples
var a = 3 + 4
var b = 4 / 2
var c = 8 % 3

O sinal de igual (=) representa atribuição em JS.

Auto-atribuições
var n= 3
n= n + 4
var b= 4
b=b*b

Como pode ver somamos o próprio valor da variável n com 4 para dar um novo valor para a varável (n=7).

Auto-atribuição Simplificadas
var n=4
n+= 4

n/=2
n**=2
n-=1

Operadores de incremento

pré-incremento
var n=4
++n
Pós-incremento
var n= 4
n++
