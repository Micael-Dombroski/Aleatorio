Estruturas de Controle

Como já sabemos, existem as seguintes estruturas de controle:

  • Sequências: A estrutura de controle mais simples, nela os passos são executados sequencialmente do ponto A até o ponto B, tendo somente uma possibilidade de caminho a ser seguido;
  • Condição: Nessa estrutura de controle, existe a opção de passos que podem ou não ser seguidos e que, de qualquer forma, levaram do ponto A ao ponto B;

Estruturas da Repetição / Looping / Laços

Assim como nas condições, temos a presença de um teste lógico, porém, se a condição for verdadeira, o programa vai executar todos os comandos dentro da estrutura de repetição até que o resultado para a condição seja falso. Quando a condição resultar em falso, o ciclo é quebrado e os comandos seguintes são executados.

While ( )

Dentre as estruturas de repetição temos a estrutura "while()" (Enquanto...Faça) em JS. Veja a sintaxe dessa estrutura de repitação a seguir:

        while (condição) {

            Bloco de Comandos a serem seguidos 

        }

Sempre que todos os comandos da estrutura de repetição while () são executados, o computador volta para o topo da estrutura e checa a condição e caso ela resulte em falso desta vez, ele avançará para os próximos passos sem passar pelos passos de dentro da estrutura de repetição.

Conhecida também como estrutura de repetição com teste lógico no início.

Do { } while ( )

Diferente da estrutura de repetição vista anteriormente, esta estrutura de repetição possui seu teste lógico no final de sua execução. A estrutura "Do While ( )" (Repita... Enquanto), possui seu teste lógico no fim de sua estrutura. Abaixo veremos a sintaxe de sua estrutura: 

        do {

            Bloco de Comandos a serem seguidos 

        } while (condição)
