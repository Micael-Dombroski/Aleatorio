Operadores relacionais

São responsáveis por fazer uma relação entre dois valores, essa relação pode resultar em Verdadeiro ou Falso. Abaixo estão esses operadores:

  • >: Maior que
  • <: Menor que
  • >=: Maior ou igual a
  • <=: Menor ou igual a
  • ==: igual a (apenas 1 sinal de igual, representa atribuição em JS)
  • !=: diferente de

Exemplos:
5 < 2 True
7 > 4 False
8 >= 8 True
9 <= 7 False
5 == 5 True
4 != 4 False
48 = 'HellTaker' False
(4*5)/2 != 8*3 True

Note que nesse último exemplo usamos os operadores relacionais e os operadores aritméticos, os operadores relacionais sempre serão executados depois dos operadores aritméticos.

Como é possível ver no penúltimo exemplo, é possível usar tanto números como strings para fazer relações com os operadores relacionais.

Operadores relacionais de identidade

São usados para que somente possam ter resultados verdadeiro para uma comparação de igualdade caso os dois elementos possuam o mesmo valor e a mesma class, ele é representado por 3 sinais seguido de igual (===), pois o JavaScript não faz a diferenciação entre classes, ele apenas analisa se os valores são iguais ou não, olhe os exemplos a seguir:
5 == 5 True
5 == '5' True
5 === '5' False
5 === 5 True
5 != '5' False
5 !== '5' True

Operadores Lógicos

  • !: Representa Negação
  • &&: Representa Conjunção
  • ||: Representa Disjunção

Exemplo de Negação: "Me empresta uma caneta que não seja vermelha?" (Nesse exemplo você pode emprestar qualquer cor de caneta desde que não seja vermelha)

Exemplo de Conjunção: "Me empresta uma caneta azul e uma vermelha?" (Nesse exemplo receber ambas é o que conta, apenas uma não atende ao pedido)

Exemplo de Disjunção: "Me empresta uma caneta azul ou uma vermelha?" (Nesse exemplo uma das opções é o suficiente, mas receber ambas também já vale)
! true > false
! false > true

O operador de negação (!) é tratado como um operador unário, ou seja, só pode receber ou o valor True ou o valor False, não ambos, caso o valor seja True então a expressão é falsa e calso o valor seja False a expressão é verdadeira.

O operador de conjunção (&&) é um operador binário, ou seja, há dois valores lógicos, um em cada lado do operador, esse valores vão dar uma resultado lógico para essa expressão. Na conjunção ambos os valores lógicos precisam ser verdadeiros para o resultado da expressão ser True.
true && true > true
true && false > false
false && true > false
false && false > false

O operador de disjunção (||) é um operador binário, ou seja, há dois valores lógicos, um em cada lado do operador, esse valores vão dar uma resultado lógico para essa expressão. Na disjunção, apenas 1 dos valores lógicos precisa ser verdadeiro para o resultado da expressão ser True.
true || true > true
true || false > true
false || true > true
false || false > false

Importante: Caso a expressão possua todos os 3 operadores, a ordem de execução sempre será primeiramente o operador de negação, seguido pelo de conjunção e por último o de disjunção.

Exemplos de fixação:
Caso você queira saber se a idade da pessoa está entre 15 e 17: idade >= 15 && <= 17
Caso você queira saber se a pessoa mora no Estado do RJ ou SP: estado == 'RJ' || estado == 'SP'
Caso você queira saber se a pessoa ganha mais que 1500 e não é homem: salário > 1500 && sexo != 'M'

Ordem de precedência dos operadores

  1. Operadores aritméticos
    1. ()
    2. **
    3. /,%,*
    4. +,-
  2. Operadores relacionais
    • Não há uma ordem de precedência entre eles, então são feitos da esquerda para a direita
  3. Operadores lógicos
    1. Negação
    2. Conjunção
    3. Disjunção

Operadores Ternários

Possue esses dois símbolos dentro de uma expressão: ? :. É chamado de ternário pois possui 3 blocos, o primeiro bloco é um teste lógico, o segundo é o que vai acontecer caso o resultado para o teste seja True, o último é o que vai acontecer caso o resultado para o teste seja False.

Teste ? true:false

Ex:
Media >= 7.0 ? "Aprovado" : "Reprovado"
Nesse exemplo, se a media for maior ou igual a 7.0 a mensagem "Aprovado" aparece, caso seja menor que 7.0, aparece a mensagem "Reprovado".

var x = 8
var res= x % 2 == 0 ? 5 : 9
Nesse exemplo, a variável res vai receber 5 ou 9, dependendo do resultado lógico gerado pela expressão, como o resultado é True, então ela receberá o valor 5.
