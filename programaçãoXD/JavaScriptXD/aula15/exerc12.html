Arrays

As Arrays são também conhecidas como vetor e variáveis compostas, pois diferentemente das variáveis comuns que só podem armazenar um valor por vez, as Arrays são capazes de armazenar múltiplos valores.

Um array é uma variável que tem vários elementos, cada elemento é composto por seu valor e por uma chave de identificação.

Para declarar uma variável composta basta criar uma variável e colocar entre '[]' os valores dela, os valores devem estar separados por uma vírgula.

Ex:


        var num = [ 5 , 8 , 4 ]
num
5 8 4
0 1 2

No exemplo acima, os quadrado negro representa a Array num, os quadrados de borda azul são seus elementos e os números laranjas são seus valores, os números vermelhos são as chaves de identificação de cada um desses elementos, como você pode notar sempre começamos a contar a partir do número 0.

Como você pode ver, nesse exemplo o Array tem apenas 3 elementos, mas caso você queira inserir um novo valor sem precisar substituir outro valor por esse novo, você pode adicionar outro elemento para o Array da seguinte forma: 


        num  [3] = 6

Aqui declaramos que a variável composta num vai receber o valor 6 no elemento que possui a chave de acesso '3' mas como essa chave não existe ela será automaticamente criada.

Caso você queira colocar um valor novo na Array sem ter que substituir outro valor e não quer ter que saber qual é a última chave de acesso existente na sua Array, você pode fazer isto: 


        num.push(7)

Desta forma você não precisa se preocupar em saber qual foi a última posição inserida na Array.

Caso você queira saber quantos elementos tem um Array basta você fazer como mostra o exemplo abaixo:


        num.length

Caso você queira que os valores fiquem organizados em ordem crescente na Array basta você usar o seguinte comando:


        num.sort()

Para buscarmos valores dentro do Vetor usamos o comando:


        num.indexOf(7)

Esse comando nos retorna a chave de acesso do valor escolhido na Array, caso o valor buscado na Array não exista o comando retornará o valor -1 que indica que não possui esse valor na Array.
