O HTML é responsável por inserir o conteúdo na nossa página web, o CSS é responsável por estilizar esse conteúdo, o deixando mais apresentável, e o JavaScript é responsável por criar interações em nossa página. HTML é uma linguagem de marcação, CSS é uma linguagem de estilos e JavaScript é uma linguagem de programação.

Os comandos em JavaScript costumam ser apresentados em letras minúsculas, existem poucas excessões em que é usado letras maiúsculas.

Podemos usar o comando "window.alert('Mensagem')" para mostrar uma janela com mensagem antes de carregar o conteúdo da página.

Podemos usar o comando "window.confirm('Mensagem')" para abrir uma janela de confirmação ao abrir o site, essa janela por padrão tem duas caixas em que o usuário pode clicar, uma escrito "OK" e outra escrito "Cancelar".

Podemos usar o comando "window.prompt('Mensagem')" para criar uma janela ao abrir a página com um campo de digitação, um exemplo seria colocar uma pergunta na janela que irá aparecer e o usuário inserir a responsta no campo de digitação da janela.
