Existem diferentes principais tipos de dados no JavaScript sendo estes: number, 'string', boolean.

Para convertermos um dado do tipo string para number podemos usar vários métodos, como por exemplo:

  • Number.parseInt(n) ou parseInt(n): onde em n inserimos o valor que queremos converter, esse primeiro comando converte números para números inteiros (Ex: 4)
  • Number.parseFloat(n) ou parseFloat(n): onde em n inserimos o valor que queremos converter, esse segundo comando converte números para números reais (Ex:5.4, o "." é chamado de ponto flutuante)

Devemos escolher de acordo com o tipo de dado desejamos, pois caso convertermos o número para inteiro não poderá ser inserido números reais.
Atualmente podemos usar um comando mais atualizado que é o: Number(n), nele inserimos o valor de n e o computador definirá se o número é inteiro ou real e faz a conver~soa para o tipo definido.

Para convertemos um número para uma string podemos usar os seguintes comandos:

  • String(n), o n representa valor o que queremos conveter para uma string.
  • n.toString(), colocamos nesse código o valor que queremos transformar em string caso seja necessário futuramente no código.

Na ultimas versões de JS podemos usar o template para escrevermos uma frase com várias variáveis, olhe abaixo um exemplo de código com concatenação e outro com template: (Assuma que os valores das variáveis nome, idade e nota são respectivamente: Gustavo, 41 e 5.5)

Código com concatenação:
'O aluno ' + nome + ' com ' + idade + ' anos tirou a nota ' + nota
Resultado exibido: O aluno Gustavo com 41 anos tirou a nota 5.5

Código com template:
`O aluno ${nome} com ${idade} anos tirou a nota ${nota}`
Note que o resultado exibido é o mesmo porém o segundo exemplo possui uma linha mais curta de código.

Para formatar strings, usando strings junto a variáveis:

  • "Eu estou apendendo s(s representa uma variável)" //Não faz interpolação
  • "Eu estou apendendo" + s //usa concatenação
  • "Eu estou aprendendo ${s}" //Usa template string

Podemos fazer outras coisas com strings, por exemplo:

  • s.lenght //revela quantos caracteres a string tem
  • s.toUpperCase() //transforma todas as letras da string para letras maiúsculas
  • s.toLowerCase() ////transforma todas as letras da string para letras minúsculas

Podemos definir quantas casas após a vírugla serão exibidas em uma variável usando o seguinte comando: nome-da-variável.toFixed(número de casas após a vírgula)

Com o comando nome-da-variável.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'}) transformamos a variável numérica em reais.
