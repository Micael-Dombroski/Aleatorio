Condições

Existem 2 tipos de condição, a simples (if) e a composta (if / else).

Condições Aninhadas

São condições compostas com outras condições dentro de sua estrutura. Ex1:

Condição 1 = (true > Condição 2 = (true > Condição 4 = (true) or (false)) or (false) ) or (false > Condição 3 > (true) or (false > Condição 5 =(true) or (false) ) )

Ex2:


    if (condição 1) {
        bloco 1
    } else {
        if (Condição 2) {
            bloco 2
        } else {
            bloco 3
        }
    }

Condição Múltipla

Essa estrutura executa blocos de acordo com respostas fixas, tendo várias possibilidades de blocos a serem executados, porém, só 1 deles será executado no final. Não é aplicavel para todo tipo de situação, porém é muito útil para situação específica.Ex: 


    switch (Expressão) {
        case valor 1:
            bloco 1
            break
        case valor 2:
            bloco 2
            break
        case valor 3:
            bloco 3
            break
        default:
            bloco 4
            break
    }

Esse estrutura não funciona com intevalos (Ex: Se 1 até 6 faça) só com valores fixos (Ex: Se 2 faça).
