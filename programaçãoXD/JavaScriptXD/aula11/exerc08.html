Sequências

São comandos que necessitam serem executados em uma ordem específica para que funcionem de forma correta. Ex: Comando 1 -> Comando 2 -> Comando 3

Condições

Condições são comandos que só são executados caso um requisito seja cumprido, Ex: Comando 1 -> Comando 2 -> Condição(Comando 2.5) -> Comando 3.
As condições geram desvios nas sequências de comandos.

Estrutura condicional "If" e ""If / Else"

É a estrutura condicional Se, que depende de um resultado para que um determinado comando seja executado o Else é o SeNão dessa condição, que é executado quando o resultado não atende os requisitos para que os comandos do If sejam executados.

If (Condição) {

Bloco de comandos Executados se o Resultado para a Condição seja Verdadeiro

} Else {
Bloco de comandos Executados se o Resultado para a Condição seja Falso

}

Tipos de Condições

Nesse arquivo serão vistos 2 tipos de Condições

  • Condição If (Se)
  • Condição If / Else (Se / SeNão)
