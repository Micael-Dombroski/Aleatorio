Iniciando estudos DOM

DOM significa Document Object Model (modelo de objetos para documentos), é um conjunto de objetos dentro do navegador que vai dar acesso aos componentes internos do website.

Árvore DOM

É essencial sabermos fazer a árvore DOM para nosso website, é nela onde fazemos um mapa dos comandos que vão ser executados nele durante a estadia do usuário.

A Árvore DOM comença pela raíz, que é chamado de window, tudo no JS está dentro desse objeto. Dentro desse objeto temos vários outros, como por exemplo, location, document e history, etc.

Dentro de document temos um outro objeto muito importante, o HTML, que abriga a parte HTML do nosso site, dentro desse objeto, temos dois objetos principais, o head e o body, representando a cabeça e o corpo do nosso código HTML respectivamente.

É importante saber que todo objeto que abriga outro é parent do objeto abrigado, e todo objeto que se abriga em outro objeto é child do objeto que o abriga.

Dentro de head temos os objetos filho (child) meta e title por exemplo, já body é objeto pai (parent) dos objetos h1, p e div por exemplo. E por fim, dentro do objeto p temos o objeto filho (child) strong.

A árvore DOM possui níveis de hierarquia, onde a raíz é o mais alto nível hirarquico, quando mais longe dela o objeto estiver, mais baixo será seu nível na hierarquia.

Exemplificando Hierarquia

  1. window
  2. document
  3. HTML
  4. body
  5. p
  6. strong

Elementos

É tudo o que aparece na árvore DOM, podemos selecionar eles para navegar dentro da árvore DOM.

Selecionando Elementos

Aqui serão mostradas 5 formas de Selecionar um Elemento, sendo elas:

  • por Marca: Esse comando está dentro de document, esse comando permite que selecionemos mais mais de um objeto (Ex: o HTML possui duas tags p), o comando é esse: getElementsByTagName()
  • por ID: getElementByID()
  • por Nome: getElementsByName()
  • por Classe: getElementsByClassName()
  • por Seletor: é uma forma nova e é mais recomendada, usamos o comando: querySelector() e querySelectorAll()
