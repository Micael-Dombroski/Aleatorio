Estruturas de repetição

A estrutura de repetição "for ( )" é uma estrutura de repetição com variável de Controle (Ex: Para X <- 1 ate 5). Abaixo temos uma sintaxe dessa estrutura:


        for (Início; teste; incrementação) {
            Bloco de comandos a serem executados
        }

Ex:


        for ( var c = 1; c <= 10; c++ ) {
            Bloco de comandos a serem executados
        }
