Git & GitHub

Gerenciador de Versões, uma software que guarda as versões do seu projeto. Um exemplo desse tipo de software é o Git. Esse tipo de software é chamado de Repositório Local.

O problema do Git é que pode haver algum problema no computador e todos os arquivos serem perdidos, para que isso não aconteça podemos colocar nosso projeto num Repositório Remoto. Como exemplo de repositório remoto é o GitHub.

O ideal para um projeto é usarmos o Git e o GitHub em conjunto para nossos projetos.
